# Notes on Topic: Terminal Formatting

## 📖 Overview
Terminal formatting enhances the user experience by providing visual cues (colors, styles) and controlling the terminal's cursor and screen state.

## 🛠️ Commands and References

### 1. ANSI Escape Codes
Directly control colors and styles via strings sent to the terminal.

- **Syntax:** `\e[<STYLE>;<COLOR>m`

- **Common Codes:**

- `0`: Reset all styles.

- `1`: **Bold**.

- `4`: __Underline__.

- `31-37`: Foreground Colors (Red, Green, Yellow, Blue, Magenta, Cyan, White).

- `41-47`: Background Colors.

### 2. The `tput` Command
A more portable way to manage terminal capabilities by looking up the current `$TERM` definition.

| Capability | Description | Example |
| :--- | :--- | :--- |
| `setaf [0-7]` | Set Foreground Color. | `tput setaf 2` (Green) |
| `setab [0-7]` | Set Background Color. | `tput setab 1` (Red) |
| `bold` / `sgr0` | Start Bold / Reset styles. | `tput bold` |
| `cup <r> <c>` | Move cursor to Row R, Col C. | `tput cup 10 20` |
| `cols` / `lines` | Get terminal width/height. | `tput cols` |
| `invis` / `cvis` | Hide / Show cursor. | `tput invis` |

### 3. Hex and ASCII Printing
Use `printf` to output non-printable characters or specific byte values.

- **Syntax:** `printf '%b' '\xNN'` (where NN is the Hex code).

- **Example:** `printf '%b' '\x41'` prints 'A'.

## 💡 Practical Use-Cases
1. **Interactive Menus:** Using `tput cup` to place options at specific locations and `tput rev` to highlight the selected item.

2. **Logging:** Using ANSI colors (Red for ERROR, Green for OK) to make logs quickly scannable.

3. **Password Security:** Using `tput invis` to hide user input during sensitive data entry.
